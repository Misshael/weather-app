# Weather App

Aplicación del clima que consulta WeatherAPI.  
Permite buscar una ciudad, ver clima actual, pronóstico de 3 días y guarda la última búsqueda en localStorage.

## Tecnologías
- HTML
- CSS
- JavaScript
- WeatherAPI

## Funcionalidades
- Búsqueda de ciudad
- Clima actual
- Pronóstico a 3 días
- Manejo de errores
- Persistencia con localStorage
- Diseño responsivo
